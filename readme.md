## Quick Start: AngularJS SPA and NodeJS

Jump start AngularJS single page application running on NodeJS server  

```
app.js
routes/
--- api.js
bin/
--- www
public/
--- app/
------ app.js
------ controllers/
--------- controller.js
------ directives/
--------- directive.js
------ services/
--------- service.js
------ views/
--------- index.html
--------- templates/
------------ template.tpl.html
--- js/
------ custom.js
--- styles/
------ custom.css
--- vendor/
------ angular-1.6.6/
------ bootstrap-3.3.7/
package.json
```

-----

### Vendor Files
- AngularJS 1.6.6
- Bootstrap 3.3.7

### Instructions
1. Download repository
2. Install dependences: ```npm install```
3. Start Server: ```npm run start```
